# Outreach Pack — 二手安心交易 MVP

正式站：`https://secondhand-safe-trade-api.smartmmmoney.workers.dev`

## 一句話定位

私下二手交易不用只靠人品：先把商品、金額、出貨、驗收與爭議規則寫進一個連結。

## 目標順序

1. FB 二手 3C / 相機 / 模型 / 公仔社團管理者。
2. LINE 代購、小賣家、團購群主。
3. 真實二手賣家：手機、相機、筆電、硬體錢包、模型。
4. 內向或不想面交喬細節的買賣雙方。

## 回覆判讀

- 「這個我會用」：邀請建立交易連結。
- 「我怕被騙」：追問最怕買方還是賣方哪一步。
- 「我不想註冊」：強調目前低門檻、只收流程回饋。
- 「我不信任 USDC/USDT」：強調目前不碰真實金流，只做流程紀錄。
- 「我願意付費」：記錄可接受價格與交易場景。
